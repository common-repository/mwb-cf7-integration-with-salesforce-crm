== WHAT IS INCLUDED IN FREE MWB CF7 INTEGRATION WITH  SALESFORCE PLUGIN == 

**1)** **Smooth CF7 Integration With Your salesforce Account**
MWB CF7 Integration with Salesforce **offers a smooth integration of both.**  It is not at all complex. The admin can enter their Salesforce API credentials to integrate Contact Form 7 with their Salesforce accounts.  
The admin can select the type of environment of your Salesforce account. Integrate either by using your own app or directly via a global app to connect with your own Salesforce account.
Enabling MWB CF7 Integration With Salesforce plugin creates a sub-menu in the name of Salesforce on the Contact Form. The admin can sign-up or log in to his Salesforce account and connect the CF7 and Salesforce   
A set of instructions are clearly mentioned on the backend for the admin to connect their Salesforce account with the Contact Form 7 plugin.

**2)** **Easy CF7 Fields Association With Any Salesforce Module Fields**
With MWB CF7 Integration With Salesforce, the admin can link **any Contact form 7 field to every available Salesforce module field.** Any module field like Contacts, Accounts, Leads, Quotes, Deals, and Sales integrates perfectly with this plugin.

**3)** **Filter CF7 Submissions According To User Input**
The admin can **filter the Contact form 7 submissions based on user input** using AND/OR logic. This logic will filter CF7 forms submissions and send it to Salesforce depending on user inputs. 
The MWB CF7 Integration With Salesforce plugin also makes wrong inputs by the user invalid, and in such a case, the plugin will not sync the invalid CF7 form entries to the Salesforce.

**4)** **Detailed Log Of CF7 Submission Sent To Salesforce**
The Salesforce CF7 integration plugin will provide **a detailed log of each Contact Form 7 entry** sent to Salesforce as per the response from Salesforce. Integrate your Contact Form 7 with Salesforce  that comes with **logging of all the API interaction with Salesforce** for better error handling

**5)** **Primary Key To Update Existing Entry Over Salesforce**
Suppose an entry made by the user in the contact form submission already exists over Salesforce. In that case, the admin can update it with the help of the **“Primary key”** provided. In this way, it also becomes effortless to integrate Contact Form 7 with Salesforce Campaigns. 

**6)** **Error eMail Notification For Admin**
MWB CF7 Integration With Salesforce plugin creates **Email notifications that are** sent to the admin if any input error occurs in the process of data, sending entries over to Salesforce. This way, the **admin gets notified of any slight error**  in real-time. 

**7)** **Collect Case and Lead through Salesforce Web to Lead and Web to Case**
Generate new cases with Web-to-Case. Quickly put up web forms to accept cases. Moreover, capture customer’s information to store as a new lead in Salesforce. If REST API is disabled for your Org (Salesforce Edition), or if you are not sure whether your org supports REST API or not, then you can authorize the plugin via web-to-lead / web-to-case

== Frequently Asked Questions ==

= What is the basic purpose of using the MWB CF7 Integration with the Salesforce plugin? =
The MWB CF7 Integration with Salesforce plugin sends all CF7 data submissions to Salesforce as per the form field mappings and classifies them on the basis of various objects and Salesforce modules assigned.

= How to Authorize the Connection of Your Salesforce Accounts? =
The admin needs to have the Salesforce account or can directly sign up using the link we have provided in our plugin in the Salesforce Account tab and then proceed with selecting between the Production and Sandbox Environment. Then enter the Consumer Key, Consumer Secret & Callback URL after generating it following the instruction shared in a plugin. The instructions can be seen by clicking the plus icon in the top right corner.

= What is the use of the conditional statement that is there in the plugin while creating the new field? =
Our plugin has a conditional filter for adding conditions to the form submissions. If the condition filters are set, only those CF7 submissions will be exported that fulfill those conditions. The admin has options for the AND filter and the OR filter.

= Can we filter Contact form 7 entries with this MWB CF7 Integration with Salesforce? = 
Yes, the CF7 Submissions filtering option is available according to the user inputs. Filtering contact forms will be made possible by Contact form 7 entries of AND / OR logic, sent to Salesforce based on user input.

= Can we update the already existing entry over Salesforce? =
Yes, the admin can update existing entries over Salesforce with the help of the “Primary key”.
